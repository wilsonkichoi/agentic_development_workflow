# Requirements: Temperature Converter

## Goal

Build a simple CLI tool that converts temperatures between Celsius and Fahrenheit.

## Must Have

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- CLI interface: `python converter.py <value> --to celsius|fahrenheit`
- Round results to 2 decimal places

## Nice to Have

- Support Kelvin (out of scope for this example)
- Interactive mode (out of scope for this example)

## Constraints

- Python only, no external dependencies
- Single file is fine for this scope
